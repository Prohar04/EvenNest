## üîß CUSTOMIZATION

### Change Colors
Edit `core/static/css/modern-design.css`:
```css
:root {
    --gradient-primary: linear-gradient(135deg, #YOUR_COLOR_1 0%, #YOUR_COLOR_2 100%);
}
```

### Change Animation Speed
```css
--micro-base: 300ms; /* Adjust milliseconds */
```

### Change Button Size
```css
.btn {
    padding: 0.75rem 1.5rem; /* Adjust padding */
}
```
